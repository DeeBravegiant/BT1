### Title
Unconstrained `fee_proposal_fri` During Startup Window Allows Proposer to Permanently Corrupt L2 Gas Price - (File: crates/apollo_consensus_orchestrator/src/validate_proposal.rs)

### Summary
During the first `fee_proposal_window_size` (10) blocks, `fee_actual` is `None` because the sliding window has not yet accumulated enough entries. In this state, `is_proposal_init_valid` skips the `fee_proposal_fri` bounds check entirely. A malicious proposer can set `fee_proposal_fri` to any arbitrary value (e.g., `u128::MAX` or `1`), which validators accept unconditionally, store in `fee_proposals_window`, and which then corrupts `fee_actual` for all blocks from height 10 onward — permanently distorting the L2 gas price for the lifetime of the chain.

### Finding Description

In `is_proposal_init_valid`, the `fee_proposal_fri` range check is guarded by an `if let` that requires both `fee_actual` and `fee_proposal` to be `Some`: [1](#0-0) 

When `fee_actual` is `None`, the entire block is skipped — no lower or upper bound is enforced on the proposer-supplied `fee_proposal_fri`.

`fee_actual` is `None` whenever `height < fee_proposal_window_size` (10), because `compute_fee_actual` performs a checked subtraction and returns `None` on underflow:

    // Validate fee_proposal is within the configured margin of fee_actual.
    // During initiation (fee_actual is None, <window_size blocks), bounds are not enforced.
    if let (Some(fee_actual), Some(fee_proposal)) =
        (proposal_init_validation.fee_actual, init_proposed.fee_proposal_fri)
    {
        let (lower_bound, upper_bound) = fee_proposal_bounds(
            fee_actual,
            VersionedConstants::latest_constants().fee_proposal_margin_ppt,
        );
        if fee_proposal.0 < lower_bound || fee_proposal.0 > upper_bound {
            return Err(ValidateProposalError::InvalidProposalInit(
                init_proposed.clone(),
                proposal_init_validation.clone(),
                format!(
                    "Fee proposal out of bounds: fee_actual={}, fee_proposal={}, allowed \
                     range=[{lower_bound}, {upper_bound}]",
                    fee_actual.0, fee_proposal.0
                ),
            ));
        }
    }
