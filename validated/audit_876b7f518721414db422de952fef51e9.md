### Title
Consensus Vote Signature Never Verified — Any Peer Can Forge Validator Votes to Force Quorum - (`crates/apollo_consensus/src/single_height_consensus.rs`)

### Summary

`handle_vote` in `SingleHeightConsensus` accepts any incoming `Vote` whose `voter` field names a committee member, without ever verifying the cryptographic signature carried in `Vote.signature`. Because the `voter` field is a plain protobuf field that any peer can set to any value, a single malicious peer can impersonate every validator in the committee, manufacture a 2/3 quorum of forged precommits, and force a `DecisionReached` event for an arbitrary block.

### Finding Description

The `Vote` struct carries a `signature: RawSignature` field alongside `voter: ContractAddress`. [1](#0-0) 

When a vote arrives over the network, `handle_vote` performs exactly two checks: height equality and committee membership of the claimed `voter` address. The signature field is never touched:

``` [2](#0-1) 

The state machine then stores the vote under the key `(round, vote.voter)` and accumulates its weight toward the quorum threshold: [3](#0-2) 

The quorum threshold is `> 2/3` of total weight (Byzantine mode): [4](#0-3) 

The self-vote path confirms that signatures are also never produced — `RawSignature::default()` (all-zero) is used: [5](#0-4) 

The verification infrastructure exists but is never called from the vote-handling path. `verify_precommit_vote_signature` is implemented in `apollo_signature_manager` and the `Staker` struct already carries the `public_key: Felt` needed for it: [6](#0-5) [7](#0-6) 

### Impact Explanation

A single peer connected to the P2P network can:

1. Enumerate the current committee (public information).
2. For each validator `V_i` with weight `w_i`, craft a `Vote { voter: V_i.address, proposal_commitment: target_block, signature: RawSignature::default(), ... }`.
3. Broadcast these forged votes. Each passes the `committee.members().iter().any(|s| s.address == vote.voter)` check.
4. The duplicate-vote guard keys on `(round, voter)`, so one forged vote per validator per round is accepted.
5. Once the accumulated weight of forged precommits exceeds `2/3` of total weight, `upon_decision` fires and emits `SMRequest::DecisionReached(Decision { precommits: forged_votes, block: target_block, ... })`.

The committed `Decision` is then handed to the orchestrator/batcher for finalization. The attacker can force commitment of any block for which a valid `ProposalCommitment` exists (including one they proposed themselves), bypassing the honest 2/3 threshold entirely.

### Likelihood Explanation

Any peer reachable by the P2P broadcast channel can trigger this. No privileged access, no stake, and no cryptographic material belonging to any validator is required. The `voter` field is a plain protobuf `Address` field decoded from the wire with no authentication binding it to the sending peer's network identity. [8](#0-7) 

### Recommendation

In `handle_vote`, after confirming committee membership, look up the staker's `public_key` from the committee and call `verify_precommit_vote_signature(block_hash, vote.signature, public_key)`. Reject the vote if verification fails or returns `false`. The `build_precommit_vote_message_digest` helper and `verify_precommit_vote_signature` are already implemented and tested in `apollo_signature_manager`.

Correspondingly, `make_self_vote` must produce a real signature using `SignatureManager::sign_precommit_vote` before broadcasting.

### Proof of Concept

```
// Attacker peer, connected to the network, committee = [V0(w=4), V1(w=3)], total=7, quorum>4.

let target = ProposalCommitment(felt!("0xdeadbeef"));

// Forge precommit from V0 (weight 4 > 4.67 threshold alone is not enough, but combined with V1 it is)
let forged_v0 = Vote {
    vote_type: VoteType::Precommit,
    height: current_height,
    round: 0,
    proposal_commitment: Some(target),
    voter: V0_address,          // committee member address, freely settable
    signature: RawSignature::default(),  // all-zero, never checked
};

// Forge precommit from V1 (weight 3, total forged weight = 7 > 4.67)
let forged_v1 = Vote { voter: V1_address, ..forged_v0 };

// Broadcast both over the consensus P2P topic.
// handle_vote accepts both (committee membership check passes).
// upon_decision fires: Decision { block: 0xdeadbeef, precommits: [forged_v0, forged_v1] }
```

The `received_vote` duplicate guard only blocks a second vote from the same `(round, voter)` pair, so one forged vote per validator per round is always accepted. [9](#0-8)

    pub(crate) fn received_vote(&self, vote: &Vote) -> VoteStatus {
        let determine_status = |old: &Vote, new: &Vote| {
            if old.proposal_commitment == new.proposal_commitment {
                VoteStatus::Duplicate
            } else {
                VoteStatus::Conflict(old.clone(), new.clone())
            }
        };

        // Check Map
        let key = (vote.round, vote.voter);
        let map_entry = match vote.vote_type {
            VoteType::Prevote => self.prevotes.get(&key),
            VoteType::Precommit => self.precommits.get(&key),
        };

        if let Some((old_vote, _)) = map_entry {
            return determine_status(old_vote, vote);
        }

        // Check Queue
        for event in &self.events_queue {
            let queued_vote = match (event, vote.vote_type) {
                (StateMachineEvent::Prevote(v), VoteType::Prevote) => v,
                (StateMachineEvent::Precommit(v), VoteType::Precommit) => v,
                _ => continue,
            };

            if queued_vote.round == vote.round && queued_vote.voter == vote.voter {
                return determine_status(queued_vote, vote);
            }
        }

        VoteStatus::New
    }

impl TryFrom<protobuf::Vote> for Vote {
    type Error = ProtobufConversionError;

    fn try_from(value: protobuf::Vote) -> Result<Self, Self::Error> {
        let vote_type = protobuf::vote::VoteType::try_from(value.vote_type)?.try_into()?;

        let height = BlockNumber(value.height);
        let round = value.round;
        let proposal_commitment: Option<ProposalCommitment> = value
            .proposal_commitment
            .map(|proposal_commitment| proposal_commitment.try_into())
            .transpose()?
            .map(ProposalCommitment);
        let voter = value.voter.ok_or(missing("voter"))?.try_into()?;
        // Convert Hashes to RawSignature (default to empty if None)
        let signature =
            value.signature.map(|hashes| hashes.try_into()).transpose()?.unwrap_or_default();

        Ok(Vote { vote_type, height, round, proposal_commitment, voter, signature })
    }
