### Title
Cairo 0 Class Declared at Future Block Served to Historical RPC Execution Without Block-Height Guard — (`crates/apollo_rpc_execution/src/state_reader.rs`)

### Summary

`ExecutionStateReader::get_compiled_class` skips the `is_contract_class_declared` check for `ContractClass::V0` (Cairo 0 / deprecated) classes when the class manager path is active. The V1 path enforces a block-height guard; the V0 path does not. The code even carries a developer TODO acknowledging the omission. Because the class manager is a block-height-agnostic cache, a V0 class declared at block N+1 is returned unconditionally when execution is requested at block N, causing RPC calls to produce wrong results.

### Finding Description

In `get_compiled_class`, when `class_manager_handle` is `Some`, the function queries the class manager and branches on the returned variant:

```
ContractClass::V1  →  is_contract_class_declared() checked  →  UndeclaredClassHash if not visible
ContractClass::V0  →  returned immediately, NO check
``` [1](#0-0) 

The developer comment at line 136 explicitly acknowledges the gap:

> `// TODO(shahak): Verify cairo0 as well after get_class_definition_block_number is fixed.`

`is_contract_class_declared` (used only for V1) queries `get_class_definition_block_number` and compares it against `self.state_number`: [2](#0-1) 

The equivalent function for deprecated classes, `get_deprecated_class_definition_block_number`, exists in storage and is used correctly by the non-class-manager fallback path (`get_contract_class`): [3](#0-2) 

The class manager (`CachedClassStorage`) stores classes by hash with no block-height metadata; `get_executable` returns whatever is cached regardless of when the class was declared: [4](#0-3) 

The RPC `call` handler constructs `ExecutionStateReader` with `class_manager_handle` set and passes `state_number` derived from the user-supplied `block_id`: [5](#0-4) 

`execute_call` wires the handle directly into the reader: [6](#0-5) 

### Impact Explanation

An RPC caller invoking `starknet_call`, `starknet_estimateFee`, or `starknet_simulateTransactions` at a historical block N receives an execution result that uses a Cairo 0 class that was not yet declared at block N. The result is authoritative-looking but wrong: the call succeeds (or produces output from the future class) when it should return `UndeclaredClassHash`. This fits the "High" impact category: *RPC execution, fee estimation, tracing, simulation, or pending view returns an authoritative-looking wrong value.*

### Likelihood Explanation

The attack requires:
1. An attacker pre-computes the class hash of a Cairo 0 class (deterministic from class code).
2. The attacker deploys a contract at block N whose code issues a `library_call` syscall with that pre-computed hash.
3. The attacker submits a `DeclareV0/V1` transaction that lands at block N+1, populating the class manager.
4. Once the node has synced past block N+1, the attacker calls `starknet_call` (or `starknet_estimateFee`) targeting block N.

All four steps are fully unprivileged: declare and deploy transactions are public operations; RPC calls require no authentication. The class hash is deterministic, so step 1 is trivial. The only prerequisite is that the node has processed block N+1 so the class manager cache is populated.

### Recommendation

Apply the same `is_contract_class_declared` pattern to the V0 branch, using `get_deprecated_class_definition_block_number` (which already exists in storage and is used correctly by the non-class-manager fallback):

```rust
ContractClass::V0(deprecated_contract_class) => {
    let deprecated_block = self
        .storage_reader
        .begin_ro_txn()
        .map_err(storage_err_to_state_err)?
        .get_state_reader()
        .map_err(storage_err_to_state_err)?
        .get_deprecated_class_definition_block_number(&class_hash)
        .map_err(storage_err_to_state_err)?;

    let is_declared = deprecated_block
        .is_some_and(|bn| self.state_number.is_after(bn));

    if is_declared {
        Ok(RunnableCompiledClass::V0(deprecated_contract_class.try_into()?))
    } else {
        Err(StateError::UndeclaredClassHash(class_hash))
    }
}
```

Remove the TODO comment once the fix is in place.

### Proof of Concept

A unit test mirroring the existing `get_compiled_class` test in `state_reader_test.rs` but for V0:

1. Create a `MockClassManagerClient` that always returns `ContractClass::V0(DeprecatedContractClass::default())` for a given hash.
2. Write storage so the deprecated class is declared at `BlockNumber(1)` (not block 0).
3. Set `state_number = StateNumber::unchecked_right_after_block(BlockNumber(0))`.
4. Call `exec_state_reader.get_compiled_class(class_hash)`.
5. **Current behaviour**: returns `Ok(RunnableCompiledClass::V0(...))` — wrong.
6. **Expected behaviour**: returns `Err(StateError::UndeclaredClassHash(class_hash))`. [7](#0-6)

pub(crate) fn is_contract_class_declared(
    txn: &StorageTxn<'_, RO>,
    class_hash: &ClassHash,
    state_number: StateNumber,

    Ok(txn
        .get_state_reader()?

pub(crate) fn get_contract_class(
    txn: &StorageTxn<'_, RO>,
    class_hash: &ClassHash,
    state_number: StateNumber,
) -> Result<Option<RunnableCompiledClass>, ExecutionUtilsError> {
    match txn.get_state_reader()?.get_class_definition_block_number(class_hash)? {
        Some(block_number) if state_number.is_before(block_number) => return Ok(None),
        Some(_block_number) => {
            let (Some(casm), Some(sierra)) = txn.get_casm_and_sierra(class_hash)? else {
                return Err(ExecutionUtilsError::CasmTableNotSynced);
            };
            let sierra_version =
                sierra.get_sierra_version().map_err(ExecutionUtilsError::SierraValidationError)?;
            return Ok(Some(RunnableCompiledClass::V1(CompiledClassV1::try_from((
                casm,
                sierra_version,
            ))?)));
        }
        None => {}
    };

    let Some(deprecated_class) =
        txn.get_state_reader()?.get_deprecated_class_definition_at(state_number, class_hash)?
    else {
        return Ok(None);
    };
    Ok(Some(RunnableCompiledClass::V0(
        CompiledClassV0::try_from(deprecated_class).map_err(ExecutionUtilsError::ProgramError)?,
    )))
}

#[test]
fn get_compiled_class() {
    let casm_contract_class = CasmContractClass {
        compiler_version: "0.0.0".to_string(),
        prime: Default::default(),
        bytecode: Default::default(),
        bytecode_segment_lengths: Default::default(),
        hints: Default::default(),
        pythonic_hints: Default::default(),
        entry_points_by_type: Default::default(),
    };
    let expected_result = casm_contract_class.clone();

    let mut mock_class_manager_client = MockClassManagerClient::new();
    mock_class_manager_client.expect_get_executable().returning(move |_| {
        Ok(Some(ContractClass::V1((casm_contract_class.clone(), SierraVersion::default()))))
    });
    let shared_mock_class_manager_client = Arc::new(mock_class_manager_client);

    // Set execution state reader with state_number to be right after block 0.
    let (mut exec_state_reader, mut storage_writer) =
        set_execution_state_reader(shared_mock_class_manager_client.clone(), 0);

    // Prepare storage so class with hash 0x2 is declared in block 0 and class with hash 0x3 is
    // declared in block 1.
    let class_hash_0x2 = class_hash!("0x2");
    let class_hash_0x3 = class_hash!("0x3");

    storage_writer
        .begin_rw_txn()
        .unwrap()
        .append_header(BlockNumber(0), &BlockHeader::default())
        .unwrap()
        .append_state_diff(
            BlockNumber(0),
            ThinStateDiff {
                class_hash_to_compiled_class_hash: indexmap!(class_hash_0x2 => compiled_class_hash!(1_u8)),
                ..Default::default()
            },
        )
        .unwrap()
        .append_header(
            BlockNumber(1),
            &BlockHeader { block_hash: BlockHash(felt!(1_u128)), ..Default::default() },
        )
        .unwrap()
        .append_state_diff(
            BlockNumber(1),
            ThinStateDiff {
                class_hash_to_compiled_class_hash: indexmap!(class_hash_0x3 => compiled_class_hash!(2_u8)),
                ..Default::default()
            },
        )
        .unwrap()
        .commit()
        .unwrap();

    // At state_number right after block 0, class with hash 0x2 is already declared.
    let result = exec_state_reader.get_compiled_class(class_hash_0x2).unwrap();
    assert_eq!(
        result,
        RunnableCompiledClass::V1(
            (expected_result.clone(), SierraVersion::default()).try_into().unwrap()
        )
    );

    // Class with hash 0x3 is not yet declared.
    let result = exec_state_reader.get_compiled_class(class_hash_0x3);
    assert_matches!(result, Err(StateError::UndeclaredClassHash(hash)) if hash == class_hash_0x3);

    // Adjust execution state reader state_number to be right after block 1.
    exec_state_reader.state_number = StateNumber::unchecked_right_after_block(BlockNumber(1));

    // Class with hash 0x2 is still declared.
    let result = exec_state_reader.get_compiled_class(class_hash_0x2).unwrap();
    assert_eq!(
        result,
        RunnableCompiledClass::V1(
            (expected_result.clone(), SierraVersion::default()).try_into().unwrap()
        )
    );

    // And class with hash 0x3 is declared too.
    let result = exec_state_reader.get_compiled_class(class_hash_0x3).unwrap();
    assert_eq!(
        result,
        RunnableCompiledClass::V1((expected_result, SierraVersion::default()).try_into().unwrap())
    );
}
