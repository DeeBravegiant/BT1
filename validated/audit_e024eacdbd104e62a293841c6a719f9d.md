Audit Report

## Title
Protocol Treasury Fee Permanently Lost on Emergency Price-Decrease Pause Early Return — (File: contracts/LRTOracle.sol)

## Summary

In `LRTOracle._updateRsETHPrice()`, when `isPriceDecreaseOffLimit` is true, the function pauses the protocol and returns early at line 281 before reaching the fee-minting block at lines 299–308. If `protocolFeeInETH > 0` was computed earlier in the same call (because TVL increased and the protocol was not yet paused), that fee is permanently unrecoverable: the subsequent manager-only price update runs with `protocolPaused == true`, forcing `protocolFeeInETH` back to zero. The treasury permanently loses the earned yield from that period.

## Finding Description

**Step 1 — Fee computation (lines 240–247):**
Before the pause check, `protocolPaused` is evaluated using the current state of `lrtDepositPool`, `withdrawalManager`, and the oracle's own `paused` flag. If none are paused and `totalETHInProtocol > previousTVL`, `protocolFeeInETH` is set to a non-zero value.

**Step 2 — Price computation (line 250):**
`newRsETHPrice = (totalETHInProtocol - protocolFeeInETH).divWad(rsethSupply)` — the fee is already subtracted from the numerator.

**Step 3 — Early return (lines 277–281):**
```solidity
if (isPriceDecreaseOffLimit) {
    if (!lrtDepositPool.paused()) lrtDepositPool.pause();
    if (!withdrawalManager.paused()) withdrawalManager.pause();
    _pause();
    return;   // <-- exits before fee minting
}
```
The function returns without updating `rsETHPrice` (line 313) or minting the fee (lines 299–308).

**Step 4 — Fee minting block never reached (lines 299–308):**
The `if (protocolFeeInETH > 0)` block that mints rsETH to `PROTOCOL_TREASURY` is skipped entirely.

**Step 5 — Next call via `updateRSETHPriceAsManager()` (lines 94–96):**
After the early return, `lrtDepositPool.paused()`, `withdrawalManager.paused()`, and `paused` are all `true`. On re-entry, line 240 sets `protocolPaused = true`, so line 244's condition `!protocolPaused && totalETHInProtocol > previousTVL` is false, and `protocolFeeInETH = 0`. The fee from the previous period is permanently unrecoverable.

No existing guard prevents this: the downside-protection block intentionally returns early, and the fee-minting block has no fallback or accumulator to retry on the next call.

## Impact Explanation

The protocol treasury (`PROTOCOL_TREASURY`) permanently loses the rsETH fee earned during the TVL-increase period immediately preceding the emergency pause. This is **permanent freezing of unclaimed yield** — yield that was earned (TVL increased, fee computed, subtracted from the price numerator) but can never be minted or recovered. This matches the allowed Medium impact: **"Permanent freezing of unclaimed yield."**

## Likelihood Explanation

The trigger is the public function `updateRSETHPrice()` (callable by any unprivileged address, subject only to `whenNotPaused` on the oracle itself — which is not yet paused at the time of the call). Three conditions must hold simultaneously:

1. `pricePercentageLimit > 0` — standard operational configuration set by admin.
2. `totalETHInProtocol > previousTVL` — staking rewards have accrued since the last price update; routine in any active protocol.
3. `newRsETHPrice` (post-fee-deduction) is still more than `pricePercentageLimit` below `highestRsethPrice` — realistic during post-slashing recovery when the all-time-high price is significantly above the current recovery level.

No attacker action is required; the scenario arises from normal protocol operation. It is repeatable across any post-slashing recovery period.

## Recommendation

Before executing the early return, mint any already-computed `protocolFeeInETH` to the treasury:

```solidity
if (isPriceDecreaseOffLimit) {
    if (protocolFeeInETH > 0) {
        uint256 feeToMint = protocolFeeInETH.divWad(newRsETHPrice);
        _checkAndUpdateDailyFeeMintLimit(feeToMint);
        if (feeToMint > 0) {
            address treasury = lrtConfig.getContract(LRTConstants.PROTOCOL_TREASURY);
            IRSETH(rsETHTokenAddress).mint(treasury, feeToMint);
            emit FeeMinted(treasury, feeToMint);
        }
    }
    if (!lrtDepositPool.paused()) lrtDepositPool.pause();
    if (!withdrawalManager.paused()) withdrawalManager.pause();
    _pause();
    return;
}
```
